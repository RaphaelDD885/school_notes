- x²=y

## Nullstellenberechnung quadratischer Funktionen

Herleitung der p-q-Formel:

f(x) = x² + px + q

0 = x² + px + q | -q

0 - q = x² + px | +(p/2)²

-q + (p/2)² = x²  + px + (p/2)² | bin. Formel 1

-q + (p/2)² = (x + p/2)² | ±√

±√-q+(p/2)² = x<sub>1|2</sub>+p/2 | -p/2

**x<sub>1|2</sub>-p/2±√(p/2)²-q**
